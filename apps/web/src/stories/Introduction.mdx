# Nexus Design System

Web Storybook for `@nexus/shared-ui` (Batch 2.6).

## How to use

1. Browse **Primitives** and **Composites** in the sidebar (Hybrid Enterprise Atomic).
2. **Patterns** and **Screens** are placeholders — no Sprint 2 implementations.
3. Use the **Theme** toolbar to switch light / dark / system.
4. Use **Controls** for args where provided.
5. Open the **Accessibility** panel (`addon-a11y`) for axe checks.

## Provider

All stories wrap production components with `SharedUIProvider` only — never import `TamaguiProvider` in stories.

## Motion

See repository doc `docs/architecture/MOTION.md`. Duration tokens + reduced-motion helpers; no animation drivers in this batch.

## Out of scope

React Native Storybook, Pattern/Screen implementations, overlays (Tooltip/Dialog/…), and product UI.

See `docs/architecture/HYBRID_ENTERPRISE_ATOMIC.md` and `docs/architecture/STORYBOOK.md`.
